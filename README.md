# Energy Australia Coding Test

Solved the question by creating a React project using Create React App and calling the API via AJAX.

### Tools Used:

- javascipt es6
- React
- Create React App
- Jest (testing)
- Babel (polyfill)
- Axios (Ajax)
- ESLint (style)
- Moxios (Axios Mocks)

### How to run project:

#### Dev mode:

```
npm start
```

#### Production Mode:

```
npm build
```

The compressed files will be available in the dist folder

### Run Tests:

```
npm test
```
